About this theme
==================
About Javascript Library used:
-----------------------------
    1. HTML5 shiv - IE to recognise and style the HTML5 elements.
                    (http://code.google.com/p/html5shiv/)
    
    2. IE9.js - JavaScript library to make Microsoft Internet Explorer behave like a standards-compliant browser.
                It fixes many HTML and CSS issues and makes transparent PNG work correctly under IE5 and IE6.
                (http://code.google.com/p/ie7-js/)
                For demo please check http://ie7-js.googlecode.com/svn/test/index.html
    
    3. CSS3 pie - PIE makes Internet Explorer 6-8 capable of rendering several of the most useful CSS3 decoration features.
                (http://css3pie.com/)

Plugin Supported:
-----------------
	1. Breadcrumb NavXT Plugin - http://wordpress.org/extend/plugins/breadcrumb-navxt/
			Go to "Dashboard >> Settings >> Breadcrumb NavXT" Settings
			
	2. WP-PageNavi (Paged Navigation) - http://wordpress.org/extend/plugins/wp-pagenavi/
			Go to "Dashboard >> Settings >> PageNavi" Settings
			
	3. Subscribe To Comments - http://txfx.net/code/wordpress/subscribe-to-comments/


Wordpress Settings::
	1. Dashboard >> Settings >> Media >> Large Size >> Max Width = 620px
	
Supported Features::
	1.	Nested Comments (Reply on comment)	
		Dashboard >> Settings >> Discussion Other comment settings 
		Check "enable threaded comments"
		
		 
	2. Google Custom Search Support
